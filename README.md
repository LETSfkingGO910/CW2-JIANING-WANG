# Laravel Task Management System
A light and heavy task management system built on Laravel + AWS EC2, implementing the core CRUD functions for tasks.
## Features
- Task list display (sorted in reverse order of creation time)
- Task creation/editing/deletion/viewing
- Task status marking (completed/incomplete)
- Form validation + operation success prompts
Deployment Environment
##
- AWS EC2 (Ubuntu 22.04)
- Laravel 11
- MySQL 8.0
- Apache 2.4
